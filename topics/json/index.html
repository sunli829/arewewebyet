JSON Support

Whether you are offering an external API or just want to talk to the modern Javascript Frontend Interface, you need to support JSON. And while there is most certainly JSON on the way, many things web-specific are still lacking.

  • serde_json

    A JSON serialization file format

  • json_in_type

    a library for fast json serialization

  • jsonway

    JSON building DSL and configurable serializers for Rust

  • jsonrpc

    Rust support for the JSON-RPC 2.0 protocol

  • iron-json-response

    Json response middleware for Iron web framework

  • json

    JSON implementation in Rust

  • valico

    JSON Schema validator and JSON coercer
