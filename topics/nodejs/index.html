Node.js

Rust can be used to write fast and safe native Node.js modules through bindings to the V8 engine!

  • cargo-node

    A cargo sub command to build, run and deploy rust wasm applications as browser, electron or cordova app.

  • neon

    A safe abstraction layer for Node.js.

  • node-bindgen

    easy way to write nodejs module using rust

  • webbundle

    WebBundle library
